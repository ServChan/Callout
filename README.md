# Callout

Client-side Fabric mod for Minecraft 26.1.2 that watches chat for important mentions, plays configurable audio alerts, and keeps a persistent history of mentions with context.

Клиентский Fabric-мод для Minecraft 26.1.2, который отслеживает важные упоминания и слова в чате, воспроизводит настраиваемые звуковые оповещения и сохраняет историю пингов с подробным контекстом.

---

## 🌟 Особенности / Features

- 🔔 **Умные триггеры и пинги / Smart Triggers:** Настраиваемые ключевые слова с поддержкой как обычного текста, так и регулярных выражений (Regex).
- 🎧 **Кастомизация звуков / Custom Audio:** Выбор любого звука Minecraft, точная настройка громкости (Volume) и высоты тона (Pitch).
- 📜 **Экран истории (`Ctrl+K`) / Mention History:** Современный тёмно-сланцевый интерфейс в стиле Glassmorphism с наглядными цветными плашками-бейджами, поддержкой колесика мыши и удобной пагинацией.
- 💬 **Расширенный контекст / Rich Context:** Сохранение 5 сообщений до и 5 сообщений после целевого пинга.
- 💾 **Сохранение на диск / Persistent Storage:** Автоматическое сохранение истории в `callout_history.json` — данные не теряются при выходе из мира или перезапуске игры.
- 🛡️ **Полная совместимость / Great Compatibility:** Умная очистка артефактов мода ChatHeads, защита от ложных срабатываний и авто-освобождение ресурсов при выключении.

---

## 🎮 Горячие клавиши / Keybindings

- `Ctrl + K` — Открыть экран истории пингов и контекста сообщений / Open ping history and context menu.
- Настройки мода доступно открыть через **Mod Menu**.

---

## 🛠️ Сборка проекта / Building

Для сборки `.jar` файла выполните в терминале:

```bash
./gradlew build
```

Готовый файл мода появится в директории `build/libs/`.
